# Mantenimiento

## update

Para buscar cambios

	sudo apt-get update

Para instalar estos cambios

	sudo apt-get upgrade

Para actualizar el sistema

	sudo apt-get dist-upgrade

Para instalar un paquete

	sudo apt-get install paquete
